## CU-08.2: Procesar Lotes de Transacciones

> **Nota técnica — Consistencia con importación individual:**
> La importación masiva reutiliza internamente `CompraInventarioService.registro_compra` y `VentaInventarioService.registrar_venta` (los mismos servicios que el endpoint individual). Esto garantiza que ambos flujos produzcan el mismo efecto sobre el stock e historial. No se incluye un caso de prueba comparativo entre ambas vías ya que dicha consistencia es una consecuencia directa de la arquitectura (mismo service llamado desde ambas vistas), no una condición a verificar en un unit test de CU-08. Si se requiere validación de extremo a extremo, deberá documentarse como test de integración en un documento separado.
